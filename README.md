# 🛒 SupriMais — Loja Virtual

E-commerce fullstack completo com carrinho de compras, checkout e pagamento integrado ao Mercado Pago.

## 🚀 Tecnologias

### Backend
- Node.js + Express 5
- PostgreSQL
- Mercado Pago SDK
- CORS + Dotenv

### Frontend
- React 19
- Vite
- React Router DOM 7
- Axios
- Tailwind CSS

## ✅ Funcionalidades

- Catálogo de produtos
- Carrinho de compras
- Checkout completo
- Pagamento real via Mercado Pago
- Painel Admin (adicionar/editar/remover produtos)
- Design responsivo (mobile-first)
- Modo escuro e claro

## 📁 Estrutura do Projeto

loja-virtual/
├── backend/      # API REST com Node.js e Express
├── frontend/     # Interface com React e Vite
└── src/          # Arquivos compartilhados

## 🔧 Como rodar localmente

### Backend
cd backend
npm install
node index.js

### Frontend
cd frontend
npm install
npm run dev

## 🔑 Variáveis de Ambiente (backend/.env)

DB_HOST=
DB_PORT=
DB_NAME=
DB_USER=
DB_PASSWORD=
